# DAF-MoE v1.5 Phase 1 Summary Tables

- 성능과 delta는 각 variant의 5개 seed JSON에 저장된 metric만 집계했다.
- 표준편차는 표본 표준편차(`ddof=1`)다.

- 학습 시간은 JSON에 저장되지 않아 각 완료 로그의 `Training Finished ... Time` 값을 집계했다. 학습이나 추론은 재실행하지 않았다.
